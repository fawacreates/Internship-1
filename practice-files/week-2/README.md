# Week 2 Practice Files

This directory contains practice exercises for Week 2: Linux & Aerospace Fundamentals

## Files

1. **linux_commands.sh** - Essential Linux commands for aerospace development
2. **coordinate_frames.py** - Python exercises on coordinate system transformations
3. **aerospace_basics.md** - Notes on aerodynamics and orbital mechanics

## Quick Start

### Run Linux Commands
```bash
bash linux_commands.sh
```

### Run Coordinate Transformation Exercises
```bash
python3 coordinate_frames.py
```
