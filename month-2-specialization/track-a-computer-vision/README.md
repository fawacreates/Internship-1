# Month 2 Track A: Computer Vision & Satellite Data

## Target Companies
Pixxel, SatSure, ideaForge, Asteria

## Learning Path (Weeks 5-8)

### Week 5: OpenCV Fundamentals
- Image processing basics
- Filtering and feature detection
- Object detection introduction
- **Resources:** [OpenCV tutorials](https://docs.opencv.org/4.x/) + PyImageSearch

### Weeks 5-6: Deep Learning for Vision
- CNNs and transfer learning
- Image classification projects
- **Resources:** [fast.ai](https://course.fast.ai/) or [CS231n](https://www.youtube.com/playlist?list=PLoROMvodv4rOc-Z5UBs2sJf6EmL5cM_pu)

### Week 6: Remote Sensing & Satellite Imagery
- Sentinel-2 and Landsat data
- Multispectral image analysis
- NDVI and vegetation indices
- **Resources:** [Google Earth Engine](https://developers.google.com/earth-engine)

### Week 7: Geospatial Data Processing
- Rasterio and GDAL usage
- GeoTIFF handling
- Coordinate transformations
- **Resources:** [Rasterio docs](https://rasterio.readthedocs.io/)

### Weeks 7-8: Applied Projects
- Land cover classification
- Crop health monitoring
- Change detection
- **Resources:** [Kaggle satellite datasets](https://www.kaggle.com/datasets?search=satellite)

## Deliverable
End-to-end trained model on satellite imagery with demonstration
