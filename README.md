# 3-Month Aerospace Internship Roadmap

**Target:** Land an internship at top aerospace companies (Bellatrix, Pixxel, Asteria, ideaForge, SatSure, Sarla Aviation, Ignis)

**Repo Structure:**
- `month-1-foundations/` - Core programming, math, electronics, and aerospace fundamentals
- `month-2-specialization/` - Deep dive into Computer Vision OR Drone Autonomy
- `month-3-projects/` - Portfolio projects and applications
- `practice-files/` - Practice scripts and learning experiments

**Daily Commitment:** 3-4 hours/day

---

## Month 1: Foundations (Weeks 1-4)

### Week 1: Programming Core + Math Refresh
- Python (data structures, OOP, NumPy)
- C/C++ fundamentals
- Linear Algebra
- Git/GitHub basics

### Week 2: Linux & Aerospace Fundamentals
- Linux basics and command line
- Flight and aerodynamics basics
- Orbital mechanics
- Coordinate systems

### Week 3: Electronics + Embedded Systems
- Basic electronics
- Arduino programming
- Aerospace sensors (IMU, GPS, Barometer)
- Serial communication (UART, I2C, SPI)

### Week 4: Control Systems + Robotics
- PID controllers
- ROS2 introduction
- MAVLink protocol
