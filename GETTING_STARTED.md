# Getting Started

Welcome to the **12-Week Aerospace Software Internship Preparation Roadmap**.

This guide answers the most common questions, explains how to begin, describes the learning path, and gives you your first assignment.

---

# Before You Begin

This roadmap is designed for students with a software background who want to prepare for internships in aerospace, robotics, computer vision, satellite data, and autonomous systems.

You do **not** need prior aerospace experience.

You only need:

- Curiosity
- Consistent effort
- A computer
- Internet connection
- Approximately 3–4 hours per day

---

# Frequently Asked Questions

## Do I Need Prior Knowledge?

No.

You do **not** need:

- Aerospace knowledge
- Advanced mathematics
- Satellite experience
- Drone experience
- Expensive hardware

You should be comfortable with basic programming or be willing to learn Python during Week 1.

---

## Do I Need to Buy Anything?

No.

### Months 1–2

Everything is software-based.

All tools are free and open source.

### Month 3 (Optional)

If you want real hardware experience:

- Quadcopter: ₹15,000–25,000
- Sensor payload: ₹5,000–10,000

These projects are optional.

## How Much Time Does This Take?

Recommended commitment:

| Day | Time |
|------|------|
| Monday – Friday | 3–4 hours/day |
| Saturday | Review (1 hour) |
| Sunday | Reflection & Planning (1 hour) |

- Approximately 20 hours/week

# Installation

Clone the repository.

```bash
git clone https://github.com/fawacreates/Internship-1.git

cd Internship-1
```

Create a virtual environment.

```bash
python -m venv venv
```

Activate it.

macOS/Linux

```bash
source venv/bin/activate
```

Windows

```bash
venv\Scripts\activate
```

Install dependencies.

```bash
pip install numpy pandas matplotlib jupyter opencv-python
```

Verify installation.

```bash
python --version
```

For complete installation instructions, see:

```
SETUP/SYSTEM_SETUP.md
```

# Your First Assignment

Create a file named:

```text
my_first_script.py
```

Write:

```python
name = input("What's your name? ")
github = input("What's your GitHub username? ")

print(f"Hello, {name}!")
print(f"GitHub: {github}")
```

Run it.

```bash
python my_first_script.py
```

Commit it.

```bash
git add my_first_script.py

git commit -m "Week 1: My first Python script"

git push
```

# Common Questions

### What if I get stuck?

- Read the documentation.
- Search online.
- Experiment with the code.
- Ask questions in programming communities.

Learning happens by solving problems.

---

### What if I only have two hours per day?

That's perfectly fine.

Complete the roadmap over six months instead of three.

Consistency matters more than speed.

---

### Is a BCA degree enough?

Yes.

Many aerospace companies hire software engineers for:

- Backend Development
- AI & Machine Learning
- Computer Vision
- Robotics Software
- Embedded Software
- Simulation
- Automation

Strong projects and problem-solving skills matter more than your degree title.
