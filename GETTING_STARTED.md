## 🎓 Quick Comparison: Track A vs Track B

### **Track A: Computer Vision & Satellite Data**

**What you'll learn:**
- Image processing
- Deep learning models
- Satellite imagery analysis
- Crop monitoring, land cover detection

**Best for:**
- BCA students (closer to your background)
- Pixxel, SatSure, ideaForge
- If you like ML/data science

**Difficulty:** ⭐⭐ Medium (easier)

**Hardware needed:** None

### **Track B: Drone Autonomy & Flight Software**

**What you'll learn:**
- Flight controller firmware
- Autonomous mission planning
- Obstacle avoidance algorithms
- State estimation with Kalman filters

**Best for:**
- ideaForge, Asteria, Sarla, Bellatrix
- If you like systems & robotics
- If you want to fly things

**Difficulty:** ⭐⭐⭐ Hard (complex)

**Hardware needed:** Optional (can simulate)

**Recommendation for BCA students:** Start with Track A. It's closer to your background and covers more company focus areas.

## 🚀 Installation (60 Minutes)

### **Super Quick Version**

```bash
# 1. Clone this repo
git clone https://github.com/fawacreates/Internship-1.git
cd Internship-1

# 2. Create virtual environment
python -m venv venv

# 3. Activate it
# On Mac/Linux:
source venv/bin/activate
# On Windows:
venv\Scripts\activate

# 4. Install Python packages
pip install numpy pandas matplotlib jupyter opencv-python

# 5. Done! You're ready
python --version
```

## 🎯 Your First Assignment

**This week, complete this:**

### **Assignment: Hello World Script**

1. Create file: `my_first_script.py`
2. Write this code:
```python
name = input("What's your name? ")
github = input("What's your GitHub username? ")
```

3. Run it:
```bash
python my_first_script.py
```

4. Expected output:
```
What's your name? Fawa
What's your GitHub username? fawacreates
Hello, Fawa!
GitHub: fawacreates
```
