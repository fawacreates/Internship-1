# Project 3: Build & Fly a Quadcopter (Optional)

## Target Companies
ideaForge, Asteria, Sarla

## Project Overview
Assemble, configure, and fly a real quadcopter with telemetry logging.

### Budget
₹15,000 - ₹25,000 (or partner with robotics club)

### Components
- Frame kit (F450 or similar)
- Flight controller (Pixhawk 4 or Cube)
- Motors & ESCs
- LiPo battery + charger
- GPS module
- Telemetry radio (optional)
- GoPro mount

### Key Activities
1. Assembly and balance
2. ArduPilot firmware flashing
3. Sensor calibration (compass, accelerometer, gyro)
4. PID tuning
5. GPS waypoint mission
6. Telemetry data logging
7. Post-flight analysis

1. Assembly documentation with photos
2. Calibration notes and parameters
3. Flight logs (BIN format)
