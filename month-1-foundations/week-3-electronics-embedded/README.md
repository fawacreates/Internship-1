# Week 3: Electronics + Embedded Systems Intro

## Topics

### 1. Basic Electronics
**Resource:** [All About Circuits](https://www.allaboutcircuits.com/)
**Tools:** Multimeter, breadboard kit

**Learning Goals:**
- Voltage, current, resistance (Ohm's Law)
- Resistors, capacitors, inductors
- LEDs, switches
- PWM (Pulse Width Modulation)

### 2. Arduino Programming
**Resource:** [Arduino Official Docs & Tutorials](https://docs.arduino.cc/)
**Tools:** Arduino Uno/Nano, Arduino IDE

**Learning Goals:**
- Setup and loop functions
- Digital/analog I/O
- Serial communication
- Interrupts

### 3. Aerospace Sensors Introduction
**Resource:** Datasheets - MPU6050, NEO-6M GPS, BMP280
**Tools:** Arduino IDE

**Learning Goals:**
- **IMU (Accelerometer/Gyroscope):** Measures orientation and acceleration
- **GPS:** Position and velocity
- **Barometer:** Altitude measurement
- **Magnetometer:** Compass heading

### 4. Serial Communication Protocols
**Resource:** [SparkFun Tutorials](https://learn.sparkfun.com/)
**Tools:** Logic analyzer (optional)

**Learning Goals:**
- UART (Serial) communication
- I2C (Inter-Integrated Circuit)
- SPI (Serial Peripheral Interface)
- When to use each

## Deliverable
**Arduino Project - IMU + GPS Logger:**
- [ ] Read data from MPU6050 IMU
- [ ] Read data from NEO-6M GPS module
- [ ] Log combined sensor data to serial monitor
- [ ] Save data to SD card (bonus)
