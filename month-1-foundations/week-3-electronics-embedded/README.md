# Week 3: Electronics + Embedded Systems Intro

## 🎯 Goals This Week

- [ ] Master basic electronics: voltage, current, resistors, circuits
- [ ] Learn Arduino programming fundamentals
- [ ] Understand aerospace sensors (IMU, GPS, barometer, magnetometer)
- [ ] Master serial communication protocols (UART, I2C, SPI)
- [ ] Build working Arduino project reading real sensor data

## 📚 Learning Resources

### Basic Electronics
- **allaboutcircuits.com:** https://www.allaboutcircuits.com/
  - Expected time: ~4 hours
  - **Why:** Foundation for all embedded work

### Arduino Programming
- **Arduino Official Docs:** https://docs.arduino.cc/
  - Expected time: ~3 hours
  - **Why:** Industry standard for prototyping

### Aerospace Sensors
- **Datasheets:** MPU6050, NEO-6M, BMP280, HMC5883L
  - Expected time: ~5 hours
  - **Why:** These are on flight controllers and drones

### Serial Communication
- **SparkFun Tutorials:** https://learn.sparkfun.com/
  - Search: UART, I2C, SPI
  - Expected time: ~3 hours

## 🔧 Hardware You'll Need

**Minimum Budget (~₹3,000–5,000):**
- Arduino Uno/Nano
- Breadboard
- Jumper wires
- Resistor assortment
- LEDs, capacitors

**Recommended Sensors (~₹2,000–3,000):**
- MPU6050 (IMU)
- NEO-6M (GPS)
- BMP280 (Barometric sensor)
- HMC5883L (Magnetometer)
