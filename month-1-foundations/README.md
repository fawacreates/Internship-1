# Month 1: Foundations (Weeks 1-4)

## Goal
Build strong CS fundamentals + aerospace basics + embedded systems intro

## Weekly Structure

### Week 1: Programming Core + Math Refresh
- Python fundamentals (data structures, control flow, functions)
- Linear algebra basics (3Blue1Brown)
- Git/GitHub essentials
- **Deliverable:** 3-4 Python scripts on GitHub ✅

### Week 2: Linux, Command Line & Aerospace Fundamentals
- Linux basics (bash, permissions, file systems)
- Flight dynamics and aerodynamics fundamentals
- Orbital mechanics basics
- Coordinate systems and reference frames
- **Deliverable:** Coordinate frames notes + aerodynamics writeup

### Week 3: Electronics + Embedded Systems Intro
- Basic electronics (circuits, breadboarding)
- Arduino programming
- Aerospace sensors (IMU, GPS, barometer datasheets)
- Serial protocols (UART, I2C, SPI)
- **Deliverable:** Arduino project reading sensors

### Week 4: Control Systems + Intro to Robotics Software
- Control systems and PID theory
- ROS2 introduction
- MAVLink protocol basics
- **Deliverable:** PID simulation + ROS2 pub/sub node
