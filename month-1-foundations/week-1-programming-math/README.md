## 📚 Learning Resources

### Python Fundamentals
- **CS50P (Harvard):** https://cs50.harvard.edu/python/
  - Lectures 0–2 cover: introduction, functions, variables, loops
  - Expected time: ~8 hours
  - **Why:** Industry-standard intro, covers everything you'll need for aerospace data processing

### C/C++ Fundamentals
- **Learn C the Hard Way:** https://learncodethehardway.org/c/
  - Focus on: pointers, memory allocation, structs
  - Expected time: ~6 hours this week
  - **Why:** Essential for embedded systems (flight controllers, firmware)

### Linear Algebra
- **3Blue1Brown "Essence of Linear Algebra":** https://www.3blue1brown.com/topics/linear-algebra
  - Watch: Chapters 1–4 (Vectors, Linear combinations, Transformations)
  - Expected time: ~3 hours
  - **Why:** Critical for computer vision, state estimation, control systems

### Git/GitHub
- **Git Immersion:** https://gitimmersion.com/
  - Complete labs 1–30
  - Expected time: ~2 hours
  - **Why:** Professional workflow, portfolio building
